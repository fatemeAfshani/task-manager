# Task Manager

this is a simple task manager application built with nest framework


after signup/signin user can create a task, change it's status, get tasks with filter, get specific task and delete it.

## Prerequisites
* node.js
* npm
* nestjs
* docker
* docker-compose

## Installation

```bash
yarn install
```

## Usage

```bash
yarn start:dev
```

## Run e2e tests
```bash
yarn test:e2e
```
